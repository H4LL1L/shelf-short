# Live Ops Plan (4 Weeks)

## Objectives
- Increase day-1 retention with daily mission loop.
- Improve session depth through lightweight economy.
- Reduce early churn with clear progression rewards.

## Week 1
- Launch baseline mission rotation.
- Track mission completion rates and average session count.
- Tune mission goals if completion rate < 35%.

## Week 2
- Add weekend coin bonus event (x1.3 win reward).
- Monitor economy inflation and booster conversion.

## Week 3
- Add limited-time challenge ladder (10 levels sprint).
- Add achievement highlight banner on home screen.

## Week 4
- Introduce themed item pack (visual-only event skin plan).
- Run A/B on extra shuffle cost (120 vs 140 coins).

## Core Metrics
- D1 retention
- D7 retention
- Average session length
- Average sessions per day
- Mission completion rate
- Booster purchase rate

## Risk Controls
- Avoid aggressive ad interrupts in core puzzle flow.
- Keep reward economy stable: no runaway inflation.
- Ensure save data reliability across app restarts.
