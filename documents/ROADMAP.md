# Implementation Roadmap

## Phase 1 - Foundation
- Create Flutter project scaffold.
- Add folder structure by feature/layer.
- Implement shared theme and constants.

## Phase 2 - Core Gameplay
- Implement domain entities and rules.
- Implement board generation and match resolution.
- Build game controller and immutable session state.

## Phase 3 - Playable UI
- Build home screen and gameplay screen.
- Build grid, tray, HUD, and action buttons.
- Add win/lose overlays and restart flow.

## Phase 4 - Polish
- Add animations and visual refinement.
- Add score tuning and difficulty curve.
- Add lightweight sound hooks.

## Phase 5 - Validation
- Unit tests for domain logic.
- Widget tests for critical flow.
- Manual playtest checklist and balancing pass.
